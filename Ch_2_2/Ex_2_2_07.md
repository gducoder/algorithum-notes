# 2.2.07

## Problem:

show that the number of compares used by merge sort is monotonically increasing (C(N+1)>C(N) for all N>0)

## Solutionï¼š

prove: C(N+1) > C(N)

if N is 2^n, then C(N) = NlgN, C(N+1) = (N+1)lg(N+1)

so C(N+1) > C(N).
