修改BinarySearch用例，删除白名单中的重复键。


## 分析：

remove duplicates in an ordered array.

```java
  public static int[] removeDuplicates(int[] a) {
    int i = 0;
    for (int j = 1; j < a.length; j++) {
      // use double pointer, i is slower than j
      // a[i] != a[j], set i moves 1, a[i] = a[j], then j moves 1
      if (a[i] != a[j]) {
        i++;
        a[i] = a[j];
      }
    }
    // all elements between 0 ~ i are distinct
    // we return a new array
    return Arrays.copyOf(a, i+1);
  }
```
